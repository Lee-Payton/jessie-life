# Jessie.Life

Rebuild of [jessie.life](https://jessie.life) — a functional-health practice for midlife women (perimenopause/menopause), metabolic testing (PNOĒ RMR + VO₂), and an education library.

## Stack
- Next.js 14 (App Router, TypeScript)
- Tailwind CSS
- Deployed on Vercel (main = production, branches = preview)

## Editing
- **Code:** this repo is the source of truth.
- **Visual editing:** connect v0.dev to this repo for AI-assisted section editing; edits sync back as commits.
